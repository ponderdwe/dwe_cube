# Contributing to DWE Cube

Thank you for your interest in contributing to DWE Cube! This project is maintained by [Ponder](https://ponder.co) and is open to contributions from school networks, ed-tech organizations, and the broader open source community.

## How to contribute

1. **Open an issue** before starting significant work — describe what you want to build or fix so we can discuss it first.
2. **Fork the repository** and create a branch from `main`.
3. **Make your changes** — keep commits focused and write clear commit messages.
4. **Open a pull request** against `main`. Describe what changed and why.

Ponder employees review and merge pull requests. We aim to respond within a few business days.

## Development setup

```bash
cp .env.example .env   # fill in your local values
docker compose up      # start Cube, dbt, Superset, and Postgres
```
